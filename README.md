# Restaurant Analytics & Recommendation Dashboard

## Project Overview

This project performs Exploratory Data Analysis (EDA) on restaurant data to identify star restaurants and generate business insights.

### Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Tableau
